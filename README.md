### You can download Expo mobile app from Google Play Market or Apple Store; scan qr code on Expo app (android), or using Camera (ios)

[React Native Self Posts](https://expo.io/@stelmakhivan/rn-self-posts)

1. Run `yarn install`
2. Run `expo start`
3. Scan qr code
